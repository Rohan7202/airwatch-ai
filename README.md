# 📖 Overview

AirWatch AI is an AI-powered environmental intelligence platform developed during the **Google Build with AI Hackathon**.

The platform enables citizens to report environmental issues such as waste accumulation and pollution using images, location, and descriptions. These reports are stored securely in Firebase and visualized on an interactive map. Google Gemini AI assists in analyzing reports and supporting prioritization, helping municipal authorities make faster and more informed decisions.

AirWatch AI demonstrates how Artificial Intelligence, cloud technologies, and geospatial visualization can improve environmental monitoring and municipal response.
